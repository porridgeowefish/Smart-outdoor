```text
前端能看到线路卡片
卡片展示封面、名称、位置、距离、爬升、展示标签，并可使用 track_preview 渲染小地图或路线轮廓
前端能获取标签 taxonomy
点击卡片进入详情
详情页能拿到 GeoJSON 轨迹用于地图渲染
详情页能拿到 visibility、primary_file 和 actions 标志
本轮不交付下载、编辑、send-to-trip-plan 接口
其他用户 private 线路不可见
```
